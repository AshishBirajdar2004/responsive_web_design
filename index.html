Introduction To C++

What is C++?

C++ is a general-purpose programming language created by Bjarne Stroustrup in 1979 at Bell Labs. It is an extension of the C programming language, adding support for advanced programming paradigms such as object-oriented programming (OOP) and features for low-level memory management. C++ combines the power and efficiency of C with modern features for software development.

Key Features

  • Object-Oriented Programming (OOP) concepts like classes and inheritance.
  • Low-level memory manipulation via pointers.
  • A rich Standard Template Library (STL).
  • Platform independence and high performance.
  • Backward compatibility with C.

Common Use Case

  • Game Development: Popular for creating high-performance games using engines like Unreal Engine.
  • System Programming: Ideal for operating systems, device drivers, and embedded systems.
  • Web Browsers: Used in the development of Chromium and Firefox.
  • Financial Applications: Preferred for trading systems and simulations because of its performance.
  • Scientific Computing: Extensively used in simulations and high-performance computing tasks.

Strucure And Syntax

Basic Structure of a C++ Program

To get started with writing C++, open a Code Editor and write your first "Hello world" C++ code. A simple C++ program typically consists of the following components:

#include // Preprocessor directive
int main() { // Main function
std::cout << "Hello, World!" << std::endl; // Output statement
return 0; // End of the program
}

Select the code in the pad and hit Ctrl+R to watch it unfold in your terminal!

Key Components

  • Preprocessor Directives:
    1. Begin with # (e.g., #include <iostream>).
    2. Used to include libraries or define macros.
    3. Example: #include <iostream> includes the Input/Output stream library.
  • Main Function:
    1. Every C++ program must have a main() function, which is the entry point.
  • Statements and Semicolons:
    1. Each statement ends with a semicolon (;).
    2. Example: std::cout << "Hello" ;
  • Braces {}:
    1. Enclose blocks of code.
  • Comments:
    1. Single-line comments: // This is a comment

Data types in C++ define the kind of data a variable can hold. They specify the type of value (e.g., integer, floating-point, character) and determine how much memory the variable occupies..C++ provides several built-in data types, classified into primitive and derived types.

Variables in C++

Variables are used to store data that can be changed during program execution.

  • Declaration and Initialization:
    • Declaration: Specifies the type and name of the variable. Ex: int age;
    • Initialization: Assigns an initial value to the variable. Ex: int age = 20;
    • Multiple Declarations: You can declare multiple variables of the same type in one line. Ex: int x = 5, y = 10, z = 15;
  • Variable Types:
    • Local Variables: Declared inside a function and accessible only within that function.
    • Global Variables: Declared outside all functions and accessible throughout the program.
    • Static Variables: Retain their value between function calls.
    • Constant Variables: Variables that cannot be modified after initialization.

Control Statements in C++

Decision-Making Statements

  • if Statement:

    An if statement is a powerful control structure used in programming to execute a specific block of code only if a particular condition evaluates to true. It's essential for decision-making in programs.

    • Condition Evaluation: The if statement checks a condition (usually a comparison or logical expression).
    • True Condition: If the condition evaluates to true, the code block within the if statement is executed.
    • False Condition: If the condition evaluates to false, the code block within the if statement is skipped, and the program continues with the next statement after the if block.
    if (x > 0) {
    std::cout << "Positive";
    }
  • if-else Statement:

    Absolutely! An if-else statement is a fundamental control structure in programming that allows you to execute different blocks of code based on whether a condition is true or false. It's a great way to handle decision-making in your program.

    • Condition Evaluation: The if statement checks a condition.
    • True Condition: If the condition evaluates to true, the code block within the if part is executed.
    • False Condition: If the condition evaluates to false, the code block within the else part is executed.
    if (x > 0) {
    std::cout << "Positive";
    } else {
    std::cout << "Non-positive";
    }
  • else if Ladder:

    The else if (or elif in Python) ladder is a way to test multiple conditions sequentially. It allows you to execute different blocks of code based on which condition is true first. If none of the conditions are true, an optional else block can be executed.

    • Condition Evaluation: The if statement checks the first condition.
    • True Condition: If the condition evaluates to true, the corresponding code block is executed, and the rest of the ladder is skipped.
    • False Condition: If the first condition is false, the else if (or elif) statement checks the next condition, and so on.
    • Final Else: If none of the conditions are true, the else block (if provided) is executed.
    if (x > 0) {
    std::cout << "Positive";
    } else if (x < 0) {
    std::cout << "Negative";
    } else {
    std::cout << "Zero";
    }
  • switch Statement:

    The switch statement is a control structure that allows you to execute one of many code blocks based on the value of a variable or expression. It's a convenient way to handle multiple possible values and their corresponding actions without writing numerous if-else statements.

    • Value Matching: The switch statement evaluates an expression or variable.
    • Case Blocks: Each possible value (called a "case") has a corresponding block of code. When a matching value is found, the associated code block is executed.
    • Default Block: If none of the cases match, an optional default block can be executed.
    switch (choice) {
    case 1: std::cout << "Option 1"; break;
    case 2: std::cout << "Option 2"; break;
    default: std::cout << "Invalid option";
    }

Looping Statements

  • for Loop:

    A for loop is a control structure used to repeat a block of code a specific number of times. It's especially useful when you know beforehand how many times you want the loop to execute.

    • Initialization: You set an initial value for a loop control variable.
    • Condition: The loop continues to execute as long as this condition evaluates to true.
    • Iteration: After each execution of the loop block, the loop control variable is updated.
    for (int i = 0; i < 5; i++) {
    std::cout << i << " ";
    }
  • while Loop:

    A while loop is a control structure used to repeat a block of code as long as a specified condition remains true. It's handy for situations where you don't know beforehand how many times you'll need to execute the loop.

    • Condition Evaluation: The condition is evaluated before each iteration of the loop.
    • True Condition: If the condition is true, the code block inside the loop is executed.
    • False Condition: When the condition becomes false, the loop stops, and the program continues with the next statement after the loop.
    int i = 0;
    while (i < 5) {
    std::cout << i << " ";
    i++;
    }
  • do-while Loop:

    The do-while loop is a control structure that guarantees the execution of a block of code at least once before checking the condition. After the first iteration, the loop continues to execute as long as the specified condition remains true. It's particularly useful when the code block needs to run at least once regardless of the condition.

    • Initial Execution: The code block inside the do part is executed once unconditionally.
    • Condition Evaluation: After executing the block, the condition is evaluated.
    • True Condition: If the condition is true, the loop continues to execute the code block.
    • False Condition: If the condition is false, the loop stops, and the program continues with the next statement after the loop.
    int i = 0;
    do {
    std::cout << i << " ";
    i++;
    } while (i < 5);

Jump Statements

  • break:

    the break statement is used to exit the nearest enclosing loop (for, while, do-while) or switch statement prematurely. Once a break statement is encountered, the program flow immediately exits that loop or switch block and continues executing the code following it.

    for (int i = 0; i < 5; i++) {
    if (i == 3) break;
    std::cout << i << " ";
    }
  • continue:

    The continue statement is used to skip the rest of the current iteration of a loop and immediately move to the next iteration. It is commonly used when a specific condition occurs, and you want to skip certain parts of the loop body without exiting the loop entirely.

    for (int i = 0; i < 5; i++) {
    if (i == 3) continue;
    std::cout << i << " ";
    }
  • return:

    The return statement is used to exit a function and optionally return a value to the calling function. It marks the end of the function's execution, and any code after the return statement within the function will not execute.

    return 0;
  • goto (Rarely Used):

    goto statement allows you to jump to a labeled statement within the same function. Although it is supported in languages like C and C++, it is rarely used because it can lead to unstructured and hard-to-read code (also called "spaghetti code"). However, it might occasionally be useful for specific scenarios like breaking out of deeply nested loops.

    goto label;
    label: std::cout << "Jumped!";

Functions in C++

What is Function in C++?

A function is a reusable block of code designed to perform a specific task, helping to organize and modularize a program.
A typical function includes:

  • A return type (data type of the value the function returns).
  • A name (identifier for the function).
  • Parameters (optional inputs to the function).
  • A body (statements defining the task).
    • int add(int a, int b) { // Function header
    return a + b; // Function body
    }

Types of Functions

Functions in programming are categorized based on their purpose, behavior, and structure. Below is an overview of types of functions commonly encountered in C++ (or similar languages):

  1. Built-in Functions:
    • Predefined in libraries, e.g., std::sqrt() (square root), std::abs() (absolute value).
    • Include headers like <cmath> or <cstdlib> to use them.
  2. User-Defined Functions:
    • Created by the programmer to perform specific tasks.
    3. float divide(int a, int b) {
    }

Function Components

  • Return type:

    Specifies the type of value the function will return (e.g., int, float, void). return type of a function specifies what type of value the function will return to the caller. This type is declared before the function's name in its definition and declaration.

    int square(int n) {
    return n * n; // Returns an integer value
    }
  • Parameters:

    Parameters are variables specified in a function's definition that act as placeholders for the values (called arguments) passed to the function when it is called. They allow functions to operate on data provided at runtime, making the function more dynamic and reusable.

    int multiply(int x, int y) { // x and y are parameters
    return x * y;
    }
  • Function Body:

    Contains the actual code to execute.The function body is the block of code enclosed within curly braces {} in a function definition. It contains the statements that define the behavior of the function—what it does when it is called.

    int subtract(int a, int b) {
    int result = a - b;
    return result;
    }

Function Declaration and Definition

  • Declaration (Prototype):

    A function declaration (prototype) is a statement that informs the compiler about the existence of a function, including its name, return type, and parameters, but without providing the implementation (function body). It allows the compiler to validate calls to the function before the actual function definition appears.

    int add(int, int); // Declaration
  • Function Definition:

    A function definition provides the actual implementation of a function—the logic or code that specifies what the function does when it is called. It includes the function name, return type, parameters, and the function body, where the implementation resides.

    int add(int a, int b) { // Definition
    return a + b;
    }

Function Call

To execute a function, call it using its name and provide necessary arguments:

int sum = add(10, 5); // Calling the add function
std::cout << "Sum: " << sum;
